## Project Summary

### Project Name
Calculator Application

### Project Description
This project is a Java-based calculator application developed as part of the 5.2 Project Application Delivery assignment. The program allows users to perform multiple mathematical operations through a menu-driven interface. The application demonstrates the ability to handle user input, perform calculations, and manage errors effectively.

### Project Tasks
- Develop a calculator application using Java
- Implement a menu-driven interface for user interaction
- Perform addition, subtraction, multiplication, and division
- Implement advanced operations such as power and square root
- Handle invalid input using exception handling
- Store and display calculation history using an ArrayList
- Test and debug the application
- Upload the completed project to GitHub

### Project Skills Learned
- Java programming fundamentals
- Methods and modular programming
- Loops and decision-making structures
- Exception handling using try-catch
- Collections using ArrayList
- Debugging and testing applications
- Version control using GitHub

### Language Used
Java

### Development Process Used
This project followed an Agile development process by building the application in phases across multiple weeks. Each phase added new features such as user interaction, calculations, and exception handling until the final version was completed.
